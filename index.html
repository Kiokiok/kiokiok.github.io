Project description

A procedural multiplayer survival adventure game. Made in Godot.

My latest and biggest project to date. I worked full time on this project for all of 2023, with the end goal of co-founding a studio. I left the team at the beginning of 2024 for personal reasons, but the project was a lot of fun to work on and taught me a lot of things. I was the sole programmer on the team, and handled pretty much every technical aspect of it. The game features an open, vertical, procedural world explorable in multiplayer. The players have many tools available to help them traverse the atypical world.

Technical information :

  • Responsible for the entirety of the programming
  • Languages used : C#, LUA, C++, GDScript

Notes :

  • Integrated Flecs on top of Godot engine
  • Built a network entity replication system on top of Enet ( creation, deletion, state change)
  • Built a custom renderer system to render entities efficiently
  • Used a data-driven approach to declare entities using the LUA scripting langage
  • Built a basic network interest manager and world streaming system
  • Built a custom entity spawning system on top of VoxelTools
  • Built an extensive state machine system for the character controller
  • Implemented the character controller and various gameplay tools
  • UI programming
